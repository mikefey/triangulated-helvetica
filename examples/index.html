Triangulated Helvetica

Eperimentation with triangulation and Helvetica using processing.js.

Usage:

Requires poly2tri.js

Include scripts

<script type='text/javascript' src='poly2tri.js'></script>
<script type='text/javascript' src='TriangulatedHelvetica.js'></script>

Create letters (currently only supports captital A-Z)

<script type='text/javascript'>
var letterTriangles = TriangulatedHelvetica.A(fontSize, x, y)
</script>
